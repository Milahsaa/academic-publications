# Overview

This research proposes a machine learning framework for cancer classification using high-dimensional microarray gene expression data.

The proposed approach applies **Haar Wavelet** for dimensionality reduction before performing classification with the **Naïve Bayes** algorithm. The study also compares the proposed method with a Genetic Algorithm (GA)-based feature selection approach.

---

# Research Motivation

Microarray datasets typically contain thousands of gene attributes but only a limited number of samples. This high-dimensional characteristic increases computational complexity and may negatively affect classification performance.

This research investigates whether Haar Wavelet-based dimensionality reduction can effectively reduce feature dimensionality while maintaining or improving classification accuracy.

---

# Methodology

The proposed framework consists of the following stages:

1. Data normalization
2. Haar Wavelet transformation for feature reduction
3. Training and testing data split
4. Naïve Bayes classification
5. Performance evaluation using classification accuracy
6. Comparison with Genetic Algorithm-based feature selection

# Results

The proposed Haar Wavelet + Naïve Bayes framework achieved competitive classification performance across multiple microarray datasets.

Key findings include:

- Effective dimensionality reduction using Haar Wavelet.
- Reduced computational complexity.
- Competitive or improved classification accuracy compared with Genetic Algorithm-based feature selection.
- Strong performance for Colon Tumor, Lung Cancer, and Ovarian Cancer datasets.
